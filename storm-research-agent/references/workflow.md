# STORM Research Workflow

Run the same workflow in Chinese, English, or bilingual mode.

## 1. Research Contract / 研究契约

State:

- Topic / 研究主题
- Decision this research supports / 本次研究服务的决策
- Available sources / 已有来源
- Unknowns that must not be guessed / 不可猜测项
- Output boundary / 输出边界

## 2. Evidence Pack / 证据包

Extract:

- Verified facts / 已验证事实
- Source path or URL for each fact / 每条事实的来源路径或 URL
- Key numbers and claims / 关键数字和主张
- Gaps and uncertain claims / 缺口和不确定主张

Label every claim:

- `sourced`
- `inferred`
- `needs verification`

Do not convert `needs verification` into fact.

## 3. Five Perspectives / 五视角

Analyze through five default perspectives:

1. Practitioner / 实践者: daily operator, real constraints, what breaks in practice
2. Academic / 学者: evidence, definitions, research quality, known limits
3. Skeptic / 怀疑者: strongest counterargument, ignored evidence, failure mode
4. Economist / 经济学家: incentives, cost, who profits, who pays
5. Historian / 历史学家: past parallels, repeated patterns, old traps

For each perspective output:

- Core position in 1-2 sentences / 核心立场
- Strongest evidence / 最强证据
- One warning only this perspective would notice / 独有提醒

## 4. Contradiction Map / 矛盾地图

List:

- Direct conflicts between perspectives / 视角之间的直接冲突
- Strongest and weakest evidence / 最强和最弱证据
- One question that would resolve the biggest contradiction / 能解决最大矛盾的问题
- What all perspectives agree on / 所有视角的共识
- What no perspective addressed / 没有任何视角覆盖的盲点

## 5. Synthesis Brief / 综合简报

Produce:

- 60-second summary / 60 秒摘要
- 5 key findings ranked by reliability / 按可靠性排序的 5 个关键发现
- Hidden connection across perspectives / 隐藏连接
- Actionable insight for the user's role / 对用户角色的可执行建议
- Frontier question / 前沿问题

## 6. Peer Review / 自我审稿

Review your own brief:

- Confidence score 1-10 for each key finding / 每个关键发现的 1-10 置信度
- Weakest claim / 最弱主张
- Evidence needed to verify it / 需要补充的证据
- Bias check / 偏差检查
- Missing sixth perspective / 缺失的第六视角
- Overall grade and what to fix / 总体评分和修正建议

## 7. Decision Card / 决策卡

End with:

- Recommendation: do / do not do / continue validating
- Biggest risk
- Smallest next experiment
- Content angle if useful
- Claims that must not be published yet

In Chinese mode, write labels as:

- 建议：做 / 不做 / 继续验证
- 最大风险
- 最小下一步实验
- 可转成内容的角度
- 暂时不能公开写的主张
